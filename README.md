Finance Dashboard:
A simple Finance Dashboard Web App built using React.js.
It helps users track their income and expenses with interactive charts and a clean user interface.

Features:
* 📊 Balance Trend Chart
* 🥧 Spending Breakdown (Pie Chart)
* 💸 Transaction Management (Add, Delete, Search, Filter)
* 🔐 Role-Based Access (Admin & Viewer)
* 🎨 Responsive and modern UI

Tech Stack:
* React.js
* Tailwind CSS
* Recharts
* Lucide React Icons
  
How to Run the Project:

1. Clone the repository:
git clone https://github.com/KavyaNaik14/finance-dashboard.git

2. Go to the project folder:
cd finance-dashboard

3. Install dependencies:
npm install

4. Start the project:
npm start

Open in browser:
[http://localhost:3000](http://localhost:3000)
